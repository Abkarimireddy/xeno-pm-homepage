# Xeno PM Intern Assignment – Homepage

This repository contains the **homepage/dashboard** designed for Xeno (getxeno.com) as part of the PM Intern Assignment.

## Overview

Xeno is a SaaS platform for retail marketers to create campaigns, manage offers, and analyze performance.  
The goal of this assignment is to define the **first screen a marketer sees after logging in**, helping them:

- Quickly understand account activity.
- Take the most impactful actions efficiently.

## How to View

- Open `home.html` in a browser.  
- Or via GitHub Pages (if enabled):  
  `https://yourusername.github.io/xeno-pm-homepage/home.html`

## Notes

- Designed with a **Product Manager perspective**—focus on clarity, actionability, and utility.
- Wireframes and detailed explanation can be submitted separately as part of the assignment.
